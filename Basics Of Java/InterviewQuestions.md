
# Java Interview Questions and Answers

### 1. What is Java?

**Answer:** Java is a high-level, object-oriented, platform-independent programming language.

### 2. Who developed Java?

**Answer:** James Gosling developed Java in 1995.

### 3. What is JVM?

**Answer:** JVM (Java Virtual Machine) executes Java bytecode.

### 4. What is JRE?

**Answer:** JRE provides the environment required to run Java programs.

### 5. What is JDK?

**Answer:** JDK contains JRE, compiler, debugger, and development tools.

### 6. Why is Java platform independent?

**Answer:** Java programs run on JVM, which is available on multiple operating systems.

### 7. What is Bytecode?

**Answer:** The intermediate code generated by the Java compiler.

### 8. What is a Class?

**Answer:** A blueprint used to create objects.

### 9. What is an Object?

**Answer:** An instance of a class.

### 10. What is OOP?

**Answer:** Object-Oriented Programming organizes programs using classes and objects.

---

### 11. What are the four pillars of OOP?

**Answer:** Encapsulation, Inheritance, Polymorphism, and Abstraction.

### 12. What is Encapsulation?

**Answer:** Wrapping data and methods into a single unit.

### 13. What is Inheritance?

**Answer:** Acquiring properties and methods from another class.

### 14. What is Polymorphism?

**Answer:** One method behaving in different ways.

### 15. What is Abstraction?

**Answer:** Hiding implementation details and showing essential features only.

### 16. What is Method Overloading?

**Answer:** Same method name with different parameters.

### 17. What is Method Overriding?

**Answer:** Redefining a parent class method in a child class.

### 18. Difference between Overloading and Overriding?

**Answer:** Overloading occurs in the same class; overriding occurs between parent and child classes.

### 19. What is a Constructor?

**Answer:** A special method used to initialize objects.

### 20. Can constructors be overloaded?

**Answer:** Yes.

---

### 21. What is a Variable?

**Answer:** A memory location used to store data.

### 22. What are Primitive Data Types?

**Answer:** byte, short, int, long, float, double, char, boolean.

### 23. What is Type Casting?

**Answer:** Converting one data type into another.

### 24. What is an Array?

**Answer:** A collection of similar data types stored in contiguous memory.

### 25. Difference between Array and ArrayList?

**Answer:** Arrays have fixed size; ArrayLists have dynamic size.

### 26. What is String?

**Answer:** A sequence of characters represented by the String class.

### 27. Why are Strings immutable?

**Answer:** For security, thread safety, and performance.

### 28. Difference between == and equals()?

**Answer:** == compares references; equals() compares content.

### 29. Difference between String, StringBuilder, and StringBuffer?

**Answer:**

* String → Immutable
* StringBuilder → Mutable, faster
* StringBuffer → Mutable, thread-safe

### 30. What is the length() method?

**Answer:** Returns the length of a string.

---

### 31. What is Exception Handling?

**Answer:** Handling runtime errors without terminating the program.

### 32. What are the keywords used in Exception Handling?

**Answer:** try, catch, finally, throw, throws.

### 33. What is a Checked Exception?

**Answer:** Checked during compilation.

### 34. What is an Unchecked Exception?

**Answer:** Checked during runtime.

### 35. What is finally block?

**Answer:** Executes whether an exception occurs or not.

### 36. What is Multithreading?

**Answer:** Executing multiple threads simultaneously.

### 37. Difference between Process and Thread?

**Answer:** Process is independent; threads share process resources.

### 38. What is Collection Framework?

**Answer:** A framework used to store and manipulate groups of objects.

### 39. What is HashMap?

**Answer:** Stores data as key-value pairs.

### 40. What is HashSet?

**Answer:** Stores unique values and does not allow duplicates.

---

### 41. Why is main() method static?

**Answer:** JVM can call it without creating an object.

### 42. Can we overload main()?

**Answer:** Yes.

### 43. Can we override static methods?

**Answer:** No.

### 44. What is the this keyword?

**Answer:** Refers to the current object.

### 45. What is the super keyword?

**Answer:** Refers to the parent class object.

### 46. Can Java support multiple inheritance?

**Answer:** Through interfaces only.

### 47. What is Garbage Collection?

**Answer:** Automatic memory cleanup of unused objects.

### 48. What is an Interface?

**Answer:** A blueprint containing abstract methods.

### 49. Difference between Abstract Class and Interface?

**Answer:** Abstract classes can have both abstract and concrete methods; interfaces mainly define contracts.

### 50. What is WORA?

**Answer:** "Write Once, Run Anywhere" — Java code can run on any platform with a JVM.


# Complex Java Interview Questions with Answers

### 1. Why is String immutable in Java?

**Answer:** To improve security, thread safety, caching, and performance. Once created, a String object cannot be modified.

---

### 2. Why is main() declared as static?

**Answer:** JVM can call it directly without creating an object of the class.

---

### 3. Can we override static methods?

**Answer:** No. Static methods belong to the class, not the object.

---

### 4. Difference between Heap Memory and Stack Memory?

**Stack**

* Stores local variables
* Faster access

**Heap**

* Stores objects
* Shared among threads

---

### 5. Difference between == and equals()?

**==**

* Compares references

**equals()**

* Compares object contents

---

### 6. Difference between String, StringBuilder, and StringBuffer?

| String    | StringBuilder | StringBuffer |
| --------- | ------------- | ------------ |
| Immutable | Mutable       | Mutable      |
| Slow      | Fastest       | Thread Safe  |

---

### 7. Why are Wrapper Classes needed?

**Answer:** Collections work with objects, not primitive data types.

Example:

```java
int → Integer
char → Character
```

---

### 8. What is Autoboxing and Unboxing?

**Autoboxing**

```java
Integer num = 10;
```

**Unboxing**

```java
int n = num;
```

---

### 9. Difference between Overloading and Overriding?

**Overloading**

* Compile-time polymorphism

**Overriding**

* Runtime polymorphism

---

### 10. Can constructors be inherited?

**Answer:** No.

---

### 11. Can constructors be overridden?

**Answer:** No.

---

### 12. Why doesn't Java support multiple inheritance through classes?

**Answer:** To avoid the Diamond Problem.

---

### 13. What is the Diamond Problem?

**Answer:** Ambiguity that occurs when multiple parent classes contain the same method.

---

### 14. Difference between Abstract Class and Interface?

| Abstract Class              | Interface        |
| --------------------------- | ---------------- |
| Can have constructors       | Cannot           |
| Can have instance variables | Only constants   |
| Partial abstraction         | Full abstraction |

---

### 15. Can we create an object of an abstract class?

**Answer:** No.

---

### 16. Why can't interface objects be created?

**Answer:** Interfaces contain incomplete implementations.

---

### 17. What is Runtime Polymorphism?

**Answer:** Method call decided during execution.

---

### 18. What is Compile-Time Polymorphism?

**Answer:** Method call decided during compilation.

---

### 19. What is the difference between final, finally, and finalize()?

**final**

* Prevent modification

**finally**

* Executes after try-catch

**finalize()**

* Invoked before garbage collection

---

### 20. What is Garbage Collection?

**Answer:** Automatic removal of unused objects from heap memory.

---

### 21. What is the difference between Checked and Unchecked Exceptions?

**Checked**

* Compile-time checking

**Unchecked**

* Runtime checking

---

### 22. Difference between throw and throws?

**throw**

```java
throw new Exception();
```

**throws**

```java
void test() throws Exception
```

---

### 23. What is Exception Propagation?

**Answer:** Exception moves from called method to calling method.

---

### 24. Can finally block execute without catch?

**Answer:** Yes.

---

### 25. Can try exist without catch?

**Answer:** Yes, if finally is present.

---

### 26. Difference between Process and Thread?

| Process         | Thread        |
| --------------- | ------------- |
| Independent     | Lightweight   |
| Separate Memory | Shared Memory |

---

### 27. What are the ways to create a thread?

1. Extend Thread class
2. Implement Runnable interface

---

### 28. Difference between start() and run()?

**start()**

* Creates new thread

**run()**

* Executes like normal method

---

### 29. What is Thread Life Cycle?

New → Runnable → Running → Waiting → Terminated

---

### 30. What is Synchronization?

**Answer:** Controls access to shared resources among threads.

---

### 31. Difference between Array and ArrayList?

**Array**

* Fixed size

**ArrayList**

* Dynamic size

---

### 32. Difference between ArrayList and LinkedList?

**ArrayList**

* Fast random access

**LinkedList**

* Fast insertion/deletion

---

### 33. Difference between HashSet and TreeSet?

**HashSet**

* Unordered

**TreeSet**

* Sorted

---

### 34. Difference between HashMap and Hashtable?

**HashMap**

* Not synchronized
* Allows null values

**Hashtable**

* Synchronized
* No null values

---

### 35. How does HashMap work internally?

**Answer:** Uses hashing and buckets to store key-value pairs.

---

### 36. What is a Deadlock?

**Answer:** Two or more threads wait indefinitely for each other.

---

### 37. What is Encapsulation and why is it important?

**Answer:** Protects data using private variables and controlled access methods.

---

### 38. What is the purpose of the this keyword?

**Answer:** Refers to the current object.

---

### 39. What is the purpose of the super keyword?

**Answer:** Refers to parent class members.

---

### 40. Explain JVM Memory Areas.

1. Method Area
2. Heap Area
3. Stack Area
4. Program Counter Register
5. Native Method Stack
